如何取得 Youtube API 金鑰

發表於 2023-09-08 分類於 技術文章 Disqus：

寫這篇的目的主要是為了解決在我的YT留言小助手上 api 配額爆掉的問題，想來想去最簡單的解法還是使用者用自己的 api 配額來抓
反正在基本使用上這是不需要收費的，但就需要做些簡單的申請，請參閱以下步驟：

首先要打開 Google Cloud Console

如果你是完全沒使用過 Google Cloud 的話，就先新建一個專案
如果你打開上面網址不是在 API和服務 這個頁面的話，請按左上角漢堡選單裡面去找找

專案名稱可以隨便打，後續完全不影響，也不用特別記得

新建好專案以後要等他轉一下，轉好後就會出現以下畫面，接著點左側的程式庫，我們要來新增 Youtube API 到專案裡面

啟用 Youtube Data API v3

打開程式庫頁面，在中間搜尋欄位搜尋 youtube，建議選項裡面就會有我們的目標 youtube data api v3，找到他後並啟用他


啟用也需要轉一下，啟用完成後會看到以下畫面，上面會有個提示說 可能需要憑證才能使用這個API
所以我們就按一下左側的憑證按鈕

選擇＋建立憑證 -> API金鑰

然後畫面上就會顯示你的API金鑰了，請複製下來做使用
在你不亂給別人這組金鑰的前提下，應該也不會有被濫用的問題，所以其他細節設定就不需要再處理，只要記好自己的API金鑰即可
我的程式只會幫你把金鑰存在你的機器上面，我這邊不會儲存你的金鑰內容
如果忘了的話也可以點擊右方的顯示金鑰


以上，其他 Google API 的申請也是一樣的方式，但一般人應該也用不到就是了。
